Stuff I built

Real-Time Bank Fraud Detection

0.89AUC-ROC 0.63KS Stat <100msLatency
Spark Scala Kafka HDFS

Built this because I was tired of seeing fraud detection demos that only work on CSV files. This thing consumes live transactions from Kafka, runs feature engineering in Spark Structured Streaming, and scores events with a GBT classifier before the user even sees their debit notification.

The 1.1% fraud rate meant I had to get creative with class weights and median imputation. AUC-ROC of 0.89 doesn't sound sexy until you realize it's on streaming data with real concept drift.

Multilingual Fake News Detector

99.71%Accuracy 0.9999AUC
FastAPI XLM-R SBERT Docker

An MLOps pipeline that actually has tests. Ingests news articles in multiple languages, generates embeddings with Sentence-BERT, and runs an ensemble of LightGBM + fine-tuned XLM-RoBERTa.

Served through a FastAPI API with request logging, model versioning, and enough Prometheus metrics to make your DevOps team cry tears of joy. The 99.71% accuracy is nice, but I'm prouder of the Dockerfile being under 200MB.

Image Super-Resolution

38.48PSNR 0.99SSIM
PyTorch Autoencoder Prometheus

Convolutional autoencoder trained on 100K real-world noisy images. Takes a grainy 48×48 thumbnail and spits out a clean 96×96 version.

Full MLOps stack because why not: FastAPI for serving, Docker Compose for orchestration, Prometheus + Grafana for monitoring, GitHub Actions for CI. Probably over-engineered for a student project. No regrets.

VITON-HD Virtual Try-On

SegFormer MediaPipe CUDA

Built a virtual try-on system using the VITON-HD architecture. Integrated SegFormer for human parsing and MediaPipe for pose estimation so the clothes actually fit the body instead of floating in space.

Had to do some sketchy memory optimizations to get it training on a single consumer GPU. Self-attention is great until you run out of VRAM.

SME Loan Default Predictor

Risk Modeling EDA Ensemble

Credit risk model on LendingClub data from 2007–2018. Compared Logistic Regression, Random Forest, and Gradient Boosting. Did way too much EDA and found that debt-to-income ratio and loan grade basically tell the whole story.

Translated the model output into a risk scoring logic that lenders could actually use. Sometimes the simplest model in production beats the fanciest one in a notebook.
