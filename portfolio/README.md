# Prit Mayani — Portfolio

Personal portfolio website built with React, TypeScript, Three.js, and GSAP.

> **Note:** Uses GSAP trial plugins for local development. For production, replace with GSAP Club plugins — see [GSAP Installation](https://gsap.com/docs/v3/Installation/).

## Tech Stack

React, TypeScript, GSAP, Three.js, WebGL, Vite

## Getting Started

```bash
npm install
npm run dev
```

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start dev server |
| `npm run build` | Production build |
| `npm run preview` | Preview production build |
| `npm run lint` | Run ESLint |
